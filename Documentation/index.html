A) Installation - top

1. Upload zipped file Livestock to your server directory and unzip there.
2. Create a database.
3. Go to the Database file application/config/database.php and fill up with your database details.

$db['default']['hostname'] = 'Host Name';
$db['default']['username'] = 'Database Username';
$db['default']['password'] = 'Password';
$db['default']['database'] = 'Database Name';


4. Import the sql file into your database from the Database folder.
5. Upload the .htaccess file from htaccess folder into the server directory.

Its Done !!!!!

B)Modules - top

Through this Farm Management User can manage administrational and financial activities. In this system there are modules like Supplier Management, Client Management, Staff Management through which user can manage the human resources and their details.

C) Sheds - top

Shed's name can be listed in Shed module.
New Sheds can be added by clicking on 'add new Shed' button.

D) Vaccination - top

Vaccination details can be listed in Vaccine module.
New Vaccination can be made by clicking on 'add new Vaccine' button.

E) Human Resources - top

Human resources comprise with three different modules

Those are:
1) Client Management
2) Supplier Management
2) Staff Management

F) Expense Management - top

Users also can entry expenses.

Steps:
1. Go to Expenses --> Expense category.
2. Create Expense categories by clicking "Add Expense Category" Button.
3. Go to Expenses --> Add Expense.
4. Select your expense category and enter the amount.

G) Sales - top

To use the Sales you need to create and maintain certain things.

Steps:
1. Create all the Product categories from Product --> Product Category at side menu.
2. Then create the product from Product-> Add product.
3. Go to Sale.
4. Select client if there is any or leave the select option.
5. Select desired product. If there are any discount input them.
6. Partial payment can be possible. In amount received field input the received amount. due will be
automatically calculated and will be shown in the invoice. 7. If any cleint wants to pay due amount. Go to the invoice.
There is a add deposit form. Input the amount client give and submit.

If the payment is successfull it will redirect to the invoice.
You can print the invoice by clicking 'print' button.

C) Products - top

Products are listed here.
New products can be created by clicking on 'add new' button.
